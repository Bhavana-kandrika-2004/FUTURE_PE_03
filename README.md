# AI SEO Blog & Content Cluster Generator

## Project Overview

This project demonstrates how AI can generate SEO-optimized blog content and content clusters for business websites.

It focuses on:

* Blog outlines
* Long-form content
* Internal linking strategies
* Local SEO optimization

##  Objectives

* Generate SEO-friendly blog content
* Apply keyword intent mapping
* Create content clusters
* Improve website visibility


## Features

✔ SEO blog outline generation (H1–H3)
✔ Long-form blog writing
✔ Internal linking strategies
✔ Local SEO optimization
✔ Business-focused content


##  Prompt Strategy

The prompt defines:

* Business details
* Keywords (main + secondary)
* Content structure
* SEO requirements

This ensures high-ranking, readable content.
